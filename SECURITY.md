# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x.x   | ✅ Yes    |
| < 1.0   | ❌ No     |

## Reporting Vulnerabilities

**Do NOT report via public issues.**

Report by:
1. GitHub private security advisory
2. Direct message to maintainer

Include:
- Issue type
- Affected code
- Reproduction steps
- Impact

## Response

- Acknowledgment: 48 hours
- Assessment: 7 days

## Disclaimer

⚠️ **Authorized testing only.**

Only scan systems you own or have permission to test.
